# TBCare — Dokumentasi Proyek

Selamat datang di dokumentasi resmi proyek TBCare. Proyek ini bertujuan untuk membangun sistem monitoring kolaboratif bagi pasien Tuberkulosis.

## Daftar Dokumen

- [**Arsitektur Aplikasi**](architecture.md): Penjelasan mengenai pola MVVM, struktur folder, dan teknologi yang digunakan.
- [**Dokumentasi Database**](database.md): Detail skema database PostgreSQL, tipe ENUM, tabel, dan relasi.

## Ringkasan Proyek

- **Platform**: Flutter (Android/iOS)
- **Backend**: Supabase (Postgres)
- **Status**: Pengembangan Aktif
- **Pola Arsitektur**: MVVM (Model-View-ViewModel)
- **Manajemen State**: Provider
- **Navigasi**: GoRouter
